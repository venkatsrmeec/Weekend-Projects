A collection of utility scripts and weekend automation tools.

## 🛠️ Latest Additions
* **optimizedlayout.py** - This file is a version of the second file controldeck.com ; this uses hotspot or shared wifi to turn android into a keyboard and other function via device browser ; no internet requred.
* **controldeck.py** - This is a maximised version of optimized layout.py , this has mroe functionalities
* **graphicstab.py** - This essentiaity turns andorid into a graphics tab like functioning , with live feed to the android device all via browser , by the means of shared wifi or hotspot ; no internet requred.
